# 放射性同位素温差发电机 (RTG Generator)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：20,000 EU
- **最大输入**：0 EU/t（不输入）
- **最大输出**：32 EU/t
- **燃料**：放射性同位素燃料靶丸

## 发电机制

### 燃料特性

- **燃料**：放射性同位素燃料靶丸（rtg_pellet）
- **无限耐久**：靶丸不消耗，可以永久使用
- **靶丸槽位**：6 个槽位

### 发电量计算

| 靶丸数量 | 发电量 |
|----------|--------|
| 0 | 0 EU/t |
| 1 | 1 EU/t |
| 2 | 2 EU/t |
| 3 | 4 EU/t |
| 4 | 8 EU/t |
| 5 | 16 EU/t |
| 6 | 32 EU/t |

### 发电量公式

```
发电量 = 2^(靶丸数量-1) EU/t
```

---

## 特殊机制

- **无限能源**：靶丸不需要补充燃料
- **可堆叠**：可以放置多个靶丸增加发电量
- **高输出**：最多可达 32 EU/t
