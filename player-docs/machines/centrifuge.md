# 热能离心机 (Thermal Centrifuge)

## 基础信息

- **电压等级**：Tier 2 (MV)
- **能量缓存**：10,000 EU
- **最大输入**：128 EU/t
- **最大输出**：0 EU/t（不输出）
- **加工耗能**：48 EU/t
- **加热耗能**：1 EU/t（不受超频影响）
- **总耗能**：49 EU/t
- **加工时间**：500 ticks (25 秒)
- **每次加工总能耗**：24,500 EU
- **热量上限**：5,000 HU
- **升级槽**：4 格（支持超频/储能/变压器）

## 热量系统

热能离心机具有热量系统：

- **热量上限**：5,000 HU
- **加热速率**：1 HU/tick（不受超频影响）
- **热量需求**：需要预热才能开始加工

## 加工条件

- 热量必须 > 0 才能开始加工
- 加热耗能即使在暂停时也会消耗
- 热量为 0 时，机器不会进行加工

## 功能

热能离心机用于分离混合流体和进行高温处理。

### 主要用途

- **生物质处理**：将生物质分离成各种流体产物
- **其他配方**：处理各种需要热量的配方

### 典型配方

| 输入 | 产出 |
|------|------|
| 生物质 | 沼气 + 蒸馏水 + 其他副产品 |

---

## 升级槽效果

| 升级类型 | 效果 |
|----------|------|
| 超频升级 | 减少加工时间 |
| 储能升级 | 增加能量缓存容量 |
| 变压器升级 | 允许接受更高电压 |
