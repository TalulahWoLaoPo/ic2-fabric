# 半流质发电机 (Semifluid Generator)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：10,000 EU
- **最大输入**：0 EU/t（不输入）
- **最大输出**：32 EU/t
- **燃料**：各种流体燃料

## 发电机制

半流质发电机使用流体燃料发电。

### 支持的流体燃料

| 流体 | 能量产出 |
|------|----------|
| 岩浆 | 20,000 EU/桶 |
| 生物燃油 | 可变 |
| 沼气 | 可变 |

### 输出特性

- **最大输出**：32 EU/t
- **流体消耗**：根据流体类型和发电量而定
