# 装罐机 (Canner)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：832 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **每 tick 能耗**：4 EU/t
- **加工时间**：200 ticks (10 秒)
- **每次加工总能耗**：800 EU
- **流体容量**：左槽 10,000 mB / 右槽 10,000 mB
- **升级槽**：4 格（支持超频/储能/变压器）

## 四种工作模式

装罐机是固体装罐机和流体装罐机的复合体，具有四种模式：

### 1. 固体装罐 (Bottle Solid)

将固体物品装入容器：
- **输入**：空锡罐 + 固体物品
- **输出**：满锡罐

### 2. 排出 (Empty Liquid)

将流体从容器排出：
- **输入**：满流体容器
- **输出**：空容器 + 流体进入机器

### 3. 灌入 (Bottle Liquid)

将流体灌入容器：
- **输入**：空容器 + 机器内有流体
- **输出**：满流体容器

### 4. 流体混合 (Enrich Liquid)

将两种流体混合（需要第二流体槽）：
- 用于特定的配方
