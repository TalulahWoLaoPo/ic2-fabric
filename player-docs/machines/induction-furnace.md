# 感应炉 (Induction Furnace)

## 基础信息

- **电压等级**：Tier 2 (MV)
- **能量缓存**：1,600 EU
- **最大输入**：128 EU/t
- **最大输出**：0 EU/t（不输出）
- **能耗**：加工时 150 EU/次 + 1 EU/t 热维持
- **加工时间**：10 ticks（热量 100% 时）
- **热量系统**：0-100%，100 秒从 0% 升温到 100%
- **升级槽**：无

## 热量系统

感应炉具有独特的热量系统：

- **热量范围**：0% - 100%
- **升温速度**：每 tick +0.05%，即 100 秒（2000 ticks）从 0% 升到 100%
- **降温速度**：热量 > 0 时，每 tick -0.05%（无燃料时自然冷却）
- **加工温度要求**：热量必须 > 0 才能开始加工

## 加工时间公式

```
实际加工时间 = 基础加工时间 / (热量% / 100)
```

| 热量 | 加工时间 |
|------|----------|
| 100% | 10 ticks (0.5秒) |
| 50% | 20 ticks (1秒) |
| 25% | 40 ticks (2秒) |
| 1% | 1000 ticks (50秒) |

## 能量消耗公式

```
总能耗 = 150 EU（固定）+ (加工 ticks × 1 EU/t 热维持)
```

| 热量 | 加工时间 | 热维持能耗 | 总能耗 |
|------|----------|------------|--------|
| 100% | 10 ticks | 10 EU | 160 EU |
| 50% | 20 ticks | 20 EU | 170 EU |
| 25% | 40 ticks | 40 EU | 190 EU |

## 双槽加工

感应炉具有两个独立的加工槽位，可以**同时加工两个物品**：

- **槽位 0**：左侧槽位
- **槽位 1**：右侧槽位
- 两个槽位共享热量系统
- 可以同时放入相同的或不同的材料

## 配方示例

感应炉支持所有原版熔炉配方，以及进阶的 IC2 专用配方。

### 矿石烧制

| 输入 | 输出 |
|------|------|
| 铁矿石 | 铁锭 |
| 金矿石 | 金锭 |

### 金属精炼（IC2 特有）

| 输入 | 输出 |
|------|------|
| 粉碎铁矿石 | 铁锭（增产） |
| 粉碎金矿石 | 金锭（增产） |
| 粉碎铜矿石 | 铜锭（增产） |
| 粉碎锡矿石 | 锡锭（增产） |
| 粉碎铅矿石 | 铅锭（增产） |
| 粉碎铀矿石 | 铀锭（增产） |

### 进阶材料

| 输入 | 输出 |
|------|------|
| 铁锭 | 精炼铁锭 |
| 锡锭 | 锡锭（净化） |

## 与普通电炉对比

| 特性 | 电炉 | 感应炉 |
|------|------|--------|
| 电压等级 | Tier 1 (LV) | Tier 2 (MV) |
| 能量缓存 | 416 EU | 1,600 EU |
| 加工时间 | 130 ticks | 10 ticks（100%热）|
| 能耗 | 3 EU/t | 150 EU/次 + 1 EU/t |
| 双槽同时加工 | 否 | 是 |
| 热量系统 | 无 | 有（0-100%）|
| 升级槽 | 0 | 0 |

---

## 特殊机制

- **热量管理**：需要时间预热，适合连续生产场景
- **双槽并行**：两个槽位同时工作，效率翻倍
- **进阶材料**：可以处理 IC2 特有的粉碎矿石，实现增产
