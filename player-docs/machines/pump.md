# 泵 (Pump)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：800 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **能耗**：20 EU/桶
- **每 tick 提取量**：1,000 mB
- **升级槽**：4 格（支持超频/储能/变压器/流体弹出/抽入）

## 功能

泵用于从流体源提取流体并输送到流体管道网络。

### 提取能力

- **每 tick 提取**：1,000 mB（0.05 桶/tick）
- **提取效率**：每秒约 1 桶

## 流体管道连接

泵需要连接到流体管道网络才能工作：

1. 泵正面连接储罐或机器
2. 泵背面连接流体管道
3. 流体从 Provider（提供方）抽取到 Receiver（接收方）

详见 [流体管道系统](../systems/fluid-pipes.md)

## 升级槽效果

| 升级类型 | 效果 |
|----------|------|
| 超频升级 | 增加提取速度 |
| 储能升级 | 增加能量缓存容量 |
| 变压器升级 | 允许接受更高电压 |
| 流体抽入升级 | 从管道网络抽入流体 |
| 流体弹出升级 | 向管道网络弹出流体 |
