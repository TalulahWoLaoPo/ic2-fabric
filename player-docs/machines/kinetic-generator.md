# 动能发电机 (Kinetic Generator)

## 基础信息

- **电压等级**：Tier 3 (HV)
- **能量缓存**：10,000 EU
- **最大输入**：0 EU/t（不输入）
- **最大输出**：512 EU/t
- **最大 KU 输入**：2,048 KU/t
- **转换比例**：4 KU = 1 EU

## 发电机制

动能发电机从传动网络接收动能（KU），并按固定比例转换为电能（EU）。

### KU 转 EU 转换

```
EU/t = KU输入 / 4
```

- 每接收 4 KU 产生 1 EU
- 最大输出 512 EU/t（需要 2,048 KU/t 输入）
- 能量缓存约等于 40,000 KU 的等效能量

### 工作阈值

| 状态 | KU 阈值 |
|------|---------|
| 启动 | ≥ 64 KU/t |
| 停止 | < 48 KU/t |

---

## 连接方式

### 动能输入

- **输入面**：正面（`canInputKuFrom`）
- 通过传动轴、传动轴网络接收 KU
- 支持水力/风力/手动动能发电机作为 KU 源

### 电能输出

- **输出面**：除正面外的五面
- 输出电压：HV（512 EU/t）
- 需要使用 HV 电缆或储电设备连接

---

## 动能来源

动能发电机需要连接到动能传动网络才能工作：

| 动能源 | 输出 KU | 说明 |
|--------|--------|------|
| [手动动能发电机](manual-kinetic-generator.md) | 取决于曲柄 | 玩家右键曲柄产生 KU |
| [水力动能发电机](water-kinetic-generator.md) | 最高 384 KU/t | 水流驱动，需转子 |
| [风力动能发电机](wind-kinetic-generator.md) | 最高 512 KU/t | 风力驱动，需转子 |

## 典型配置

### 基础配置

```
手动动能发电机 → 传动轴 → 动能发电机 → 电缆 → 储电设备
```

### 高效配置

```
风力动能发电机 → 传动轴网络 → 动能发电机 → MFE/MFSU
```
