# 作物监管机 (Cropmatron)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：10,000 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **流体槽**：水 2,000 mB / 抗草剂 2,000 mB
- **升级槽**：支持储能升级

## 功能

作物监管机管理 9×9 区域的作物，自动进行以下操作：

### 自动功能

| 功能 | 描述 |
|------|------|
| 浇水 | 使用水桶为耕地浇水 |
| 施肥 | 使用肥料催熟作物 |
| 抗草 | 使用 weed-EX 除草 |

### 流体需求

- **水**：用于浇水
- **抗草剂 (Weed-EX)**：用于除草

---

## 与作物收获机对比

| 特性 | 作物收获机 | 作物监管机 |
|------|------------|------------|
| 主要功能 | 收获作物 | 管理作物（浇水/施肥/除草） |
| 管理范围 | 单一作物 | 9×9 区域 |
| 流体需求 | 无 | 水 + 抗草剂 |
