# 洗矿机 (Ore Washing Plant)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：16,000 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **每 tick 能耗**：16 EU/t
- **加工时间**：500 ticks (25 秒)
- **每次加工总能耗**：8,000 EU
- **每次加工水耗**：1,000 mB（1 桶）
- **升级槽**：4 格（支持超频/储能/变压器）

## 功能

洗矿机使用水流清洗粉碎矿石，产出纯净粉末和副产品。

### 加工流程

1. 将粉碎矿石放入输入槽
2. 机器消耗 1,000 mB 水
3. 25 秒后完成加工
4. 产出纯净粉末和炉渣

### 产出物

| 输入 | 主要产出 | 次要产出 |
|------|----------|----------|
| 粉碎铁矿石 | 纯净铁粉 | 炉渣 |
| 粉碎金矿石 | 纯净金粉 | 炉渣 |
| 粉碎铜矿石 | 纯净铜粉 | 炉渣 |
| 粉碎铅矿石 | 纯净铅粉 | 炉渣 |
| 粉碎锡矿石 | 纯净锡粉 | 炉渣 |
| 粉碎铀矿石 | 纯净铀粉 | 炉渣 |

### 与粉碎对比

| 流程 | 产出 |
|------|------|
| 铁矿石 → 粉碎机 → 粉碎铁矿石 → 洗矿机 → 纯净铁粉 | 增产 |
| 铁矿石 → 电炉 → 铁锭 | 基础产出 |

洗矿后再冶炼可以获得更多金属。

## 升级槽效果

| 升级类型 | 效果 |
|----------|------|
| 超频升级 | 减少加工时间 |
| 储能升级 | 增加能量缓存容量 |
| 变压器升级 | 允许接受更高电压 |
