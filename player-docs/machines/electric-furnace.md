# 电炉 (Electric Furnace)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：416 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **每 tick 能耗**：3 EU/t
- **加工时间**：130 ticks (6.5 秒)
- **每次加工总能耗**：390 EU
- **升级槽**：无

## 配方

电炉支持原版 Minecraft 中所有可以在熔炉中烧制的配方。

### 矿石烧制

| 输入 | 输出 | 备注 |
|------|------|------|
| 铁矿石 | 铁锭 | - |
| 金矿石 | 金锭 | - |
| 铜矿石 | 铜锭 | - |
| 铁矿石（深板岩） | 铁锭 | - |
| 金矿石（深板岩） | 金锭 | - |
| 铜矿石（深板岩） | 铜锭 | - |

---

## 特殊机制

- **无升级槽**：电炉是基础机器，不支持超频等升级
- **电池供电**：可在电池槽放置充电电池为机器供电
- **单槽操作**：一次只处理一个物品
