# 磁化机 (Magnetizer)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：10,000 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **基础高度**：20 格
- **每超频等级增加**：4 格高度
- **脉冲能量消耗**：100 + 15 × 高度

## 功能

磁化机用于磁化铁块，使其具有吸引物品的能力。

### 工作机制

1. **放置铁块**：将铁块放在磁化机上方的特定位置
2. **激活磁化**：消耗能量脉冲磁化铁块
3. **吸引物品**：磁化后的铁块会吸引附近掉落物品

### 脉冲参数

| 参数 | 值 |
|------|------|
| 基础脉冲能量 | 100 EU |
| 每高度增加 | 15 EU |
| 脉冲间隔 | 10 ticks |
| 脉冲持续 | 10 ticks |

### 高度计算

```
有效高度 = 基础高度(20) + 超频等级 × 4
```

| 超频等级 | 有效高度 |
|----------|----------|
| 0 | 20 格 |
| 1 | 24 格 |
| 2 | 28 格 |
| 3 | 32 格 |
| 4 | 36 格 |
