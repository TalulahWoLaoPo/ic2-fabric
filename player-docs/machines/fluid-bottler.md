# 流体装罐机 (Fluid Bottler)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：250 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **每 tick 能耗**：2 EU/t
- **加工时间**：100 ticks (5 秒)
- **每次加工总能耗**：200 EU
- **流体容量**：10,000 mB
- **升级槽**：4 格（支持超频/储能/变压器）

## 功能

流体装罐机用于将流体装入容器或从容器中抽出流体。

### 灌装模式

将流体从机器内部灌入空容器：
- **输入**：空单元 / 空桶
- **输出**：装有对应流体的单元 / 桶

### 排出模式

将流体从满容器抽出到机器内部：
- **输入**：装有流体的单元 / 桶
- **输出**：空单元 / 空桶

## 流体类型

支持的流体包括：
- 水
- 岩浆
- 冷却液
- 热冷却液
- 生物燃油
- 生物质
- 沼气
- 蒸馏水
- 以及其他 IC2 相关流体
