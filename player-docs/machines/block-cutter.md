# 方块切割机 (Block Cutter)

## 基础信息

- **电压等级**：Tier 1 (LV)
- **能量缓存**：3,600 EU
- **最大输入**：32 EU/t
- **最大输出**：0 EU/t（不输出）
- **每 tick 能耗**：4 EU/t
- **加工时间**：450 ticks (22.5 秒)
- **每次加工总能耗**：1,800 EU

## 功能

方块切割机使用锯片将方块切割成板材。

### 主要用途

将方块切割成更小的板材：
- **输入**：方块（如铁块）
- **输出**：板材 ×9

## 锯片耐久

方块切割机使用锯片作为切割工具，锯片会消耗耐久：

### 锯片类型

| 锯片 | 耐久 |
|------|------|
| 铁切割锯片 | 有限 |
| 钢切割锯片 | 更长 |
| 钻石切割锯片 | 最长 |

### 耐久显示

- 当锯片过弱时，机器会显示警告提示
- 需要定期更换锯片
