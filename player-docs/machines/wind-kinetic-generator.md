# 风力动能发电机 (Wind Kinetic Generator)

## 基础信息

- **电压等级**：Tier 1
- **输出**：动能（KU）
- **输出面**：背面（正面朝向的对面）
- **转子槽**：1 格

## 功能

风力动能发电机利用风力驱动转子产生动能（KU），发电量取决于高度、天气和转子类型。

## 工作条件

### 高度影响

风力强度随高度变化，使用高斯分布模型：

```
平均风力 = 0.03 + 0.97 × exp(-(高度 - 150)² / (2 × 35²))
```

| 高度 (Y) | 平均风力系数 |
|----------|--------------|
| 64 | 约 3.7% |
| 100 | 约 18% |
| 150 | 100%（峰值） |
| 200 | 约 18% |
| 250 | 约 3.7% |

### 天气影响

| 天气 | 倍率 |
|------|------|
| 晴天 | 1.0x |
| 雨天 | 1.2x |
| 雷雨 | 1.5x |

### 阵风波动

- **阵风范围**：0.5x - 1.5x
- **更新间隔**：200 ticks（10 秒）
- **每个区块**：独立随机阵风值

## 转子类型

| 转子材质 | 半径 | 倍率 | 峰值输出（150Y，晴天） |
|----------|------|------|-------------------------|
| 木转子 | 2.0 格 | 1x | 128 KU/t |
| 铁转子 | 3.0 格 | 2x | 256 KU/t |
| 钢转子 | 4.0 格 | 3x | 384 KU/t |
| 碳转子 | 5.0 格 | 4x | 512 KU/t |

## 输出计算

### 完整公式

```
KU/t = 128 × 转子倍率 × 平均风力 × 阵风 × 天气倍率
```

### 示例计算

**钢转子，Y=150，晴天，阵风 1.0x：**
```
128 × 3 × 1.0 × 1.0 × 1.0 = 384 KU/t
```

**钢转子，Y=150，雷雨，阵风 1.3x：**
```
128 × 3 × 1.0 × 1.3 × 1.5 = 748 KU/t
```

## 转子磨损

转子在旋转时会逐渐磨损，寿命耗尽后转子消失：

### 磨损速率

| 条件 | 磨损速率 |
|------|----------|
| 晴天 | 1 点/tick |
| 雨天 | 1.2 点/tick |
| 雷雨 | 1.5 点/tick |

| 转子 | 总耐久（晴天） | 工作时间（晴天） |
|------|----------------|-----------------|
| 木转子 | ~18,000 点 | ~2.5 小时 |
| 铁转子 | ~36,000 点 | ~5 小时 |
| 钢转子 | ~54,000 点 | ~7.5 小时 |
| 碳转子 | ~72,000 点 | ~10 小时 |

## 安装建议

### 最佳高度

- **理想高度**：Y = 150（±35）
- **最低推荐**：Y ≥ 100
- **避免**：Y < 80（风力太弱）

### 注意事项

- **碰撞检测**：转子被方块阻挡会停止工作
- **高度限制**：避免放置在地面或地下
- **天气依赖**：雷雨天发电最多，但磨损也更快

## 与水力动能发电机对比

| 特性 | 水力动能发电机 | 风力动能发电机 |
|------|----------------|----------------|
| 环境要求 | 浸没在水中 | 高空（Y≈150） |
| 基础输出 | 64 KU/t | 128 KU/t |
| 加成条件 | 流水 1.5x | 高度/天气/阵风 |
| 转子半径 | 1.0-2.5 格 | 2.0-5.0 格 |
| 磨损速率 | 2-3 点/tick | 1-1.5 点/tick |
| 稳定性 | 稳定 | 波动较大 |
