## 计算说明

- **电源槽**：左边缘 8，与三槽同排 y=26
- **左储罐**：8+18=26，宽 6，高 52
- **三槽**：44, 62, 80，间距 18
- **右储罐**：80+18+6=104（留 6px 间隙避免与输出槽重叠）
- **进度条**：与输入槽左对齐 x=44，在槽下方 26+18+4=48
